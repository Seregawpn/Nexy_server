# PRODUCT_CONCEPT (СЕРВЕР)

Системный обзор (клиент + сервер) — `Docs/SYSTEM_CONCEPT.md`.

## Роль сервера

- Принимать gRPC запросы от клиента.
- Обрабатывать текст/контекст через доменные модули.
- Стримить ответы обратно клиенту (аудио/текст).

## Feature Flags (Сервер)

- Поддержка модульной сборки системного промпта (в зависимости от включенных фич).
- Валидация команд для предотвращения галлюцинаций.
- Возможность отключения подсистем: `MESSAGES`, `BROWSER`, `PAYMENT`.
