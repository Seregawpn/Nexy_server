# Archive Docs Policy

Папка `server/Docs/_archive/` содержит исторические материалы.

## Правило

Документы из этой папки:
- не являются каноническими источниками
- используются только для исторического контекста и расследований

## Актуальные каноны

- `server/Docs/SERVER_DEPLOYMENT_GUIDE.md` — деплой серверного кода (`Nexy_server`)
- `server/Docs/RELEASE_AND_UPDATE_GUIDE.md` — публикация клиентских артефактов (`Nexy_production/releases`)
- `server/Docs/DEPLOY_INCIDENT_RUNBOOK.md` — инциденты, зависимости, конфиги, rollback
- `server/Docs/ARCHITECTURE_OVERVIEW.md` — архитектурные границы и source of truth

## Использование архива

Перед применением любой команды из архива нужно сверить ее с актуальными канонами.
