# Intent Keywords Config (YAML)

## Метаданные
- Ассистент: codex
- Тип: analysis
- Дата: 2026-02-03
- ID (INS-###): INS-005

## Diagnosis
Требовалось вынести ключевые слова интентов из кода в конфиг и добавить загрузку.

## Root Cause
Словарь ключевых слов был захардкожен в prompts.py, что мешало изменению без правки кода.

## Optimal Fix
Добавлен YAML конфиг ключевых слов и загрузчик с кешем и fallback на дефолты.

## Verification
- При наличии server/config/intent_keywords.yaml используется он.
- При отсутствии файла используется дефолтный словарь.

## Запрос/цель
Сделать конфиг для ключевых слов (yaml/env) и загрузку.

## Контекст
- Файлы: server/config/prompts.py, server/config/intent_keywords.yaml
- Документы: Docs/ARCHITECTURE_OVERVIEW.md
- Ограничения: только английские ключевые слова

## Решения/выводы
- В prompts.py добавлены load_prompt_keywords() и кеш.
- Ключевые слова вынесены в server/config/intent_keywords.yaml.
- Добавлен env PROMPT_KEYWORDS_PATH для переопределения пути.
