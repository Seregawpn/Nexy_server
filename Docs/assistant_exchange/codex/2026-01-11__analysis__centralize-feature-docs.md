## Diagnosis
Процесс добавления функционала был размазан по нескольким документам с риском рассинхрона.

## Root Cause
Дублирующие описания маршрута (client/server) в разных местах → потенциальный конфликт при будущих изменениях.

## Optimal Fix
Централизовать детальные шаги в cookbook и оставить краткие ссылки/маршруты в остальных документах.

## Verification
Краткие маршруты в обзорах и CRM указывают на cookbook как источник истины.

## Запрос/цель
Сделать единую точку правды по добавлению функционала.

## Решения/выводы

- Остальные документы оставляют только краткий путь и ссылку.
