## 🤝 **ВКЛАД В РАЗВИТИЕ**

### **Как добавить новое правило:**
1. Создайте issue с описанием проблемы
2. Предложите решение в виде принципа
3. Создайте PR с новым документом
4. Обсудите с командой и внесите правки
5. После одобрения - документ добавляется в папку

### **Как улучшить существующее правило:**
1. Найдите проблему в текущем правиле
2. Предложите улучшение
3. Создайте PR с изменениями
4. Обновите версию документа
