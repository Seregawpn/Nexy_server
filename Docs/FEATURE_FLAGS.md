# Feature Flags (Root Index)

Root-документ является индексом.

Канонические реестры feature flags:
- Клиент: `client/Docs/FEATURE_FLAGS.md`
- Сервер: `server/server/Docs/FEATURE_FLAGS.md`

Связанные документы:
- Клиентская архитектура: `client/Docs/ARCHITECTURE_OVERVIEW.md`
- Серверные правила разработки: `server/server/Docs/SERVER_DEVELOPMENT_RULES.md`

Правило централизации:
- Новые флаги добавляются только в канонический реестр владельца (client/server).
- Root не дублирует значения/описания флагов.
