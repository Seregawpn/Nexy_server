# Manual Diagnostics (Legacy)

Эти скрипты выведены из активного `scripts/` контура, потому что:
- не участвуют в CI;
- не входят в preflight/release пайплайн;
- предназначены для ручной локальной диагностики.

Статус:
- Не удалены, сохранены для истории и точечной ручной отладки.
- Любой повторный ввод в активный контур должен сопровождаться ссылкой в `Docs/PROJECT_REQUIREMENTS.md` и/или подключением в CI/preflight.
