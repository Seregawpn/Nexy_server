- корректному обновлению версии клиента;
- корректной заливке клиента в GitHub.

   - Содержит:
     - Source of Truth для версии (`config/unified_config.yaml`);

     - пошаговую верификацию (поиск старой/новой версии);
     - строгую политику заливки (только `Nexy_client_test`);
     - DoD.

2. Обновлен индекс документации:

3. Обновлены клиентские правила:
   - `AGENTS.md`
   - В контекст добавлена ссылка на регламент версии/заливки.

4. Усилен релизный чеклист:
   - `RELEASE_CHECKLIST.md`
