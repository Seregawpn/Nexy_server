## Решения/выводы
- Найдены логические проблемы с условным созданием DMG/PKG и последующими обязательными проверками.
- DMG ошибочно завязан на Developer ID Installer, хотя должен требовать только Developer ID Application.
- Финальные проверки и логирование используют DMG/PKG пути без guard, что ломает сценарий без Installer сертификата.
